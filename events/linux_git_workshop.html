Session 1: Linux Fundamentals

The first session, led by Adithyan P, focused on the Linux kernel, popular distributions, and the history of the GNU Project. He highlighted the importance of operating a Linux system without relying on a GUI, showing how to work entirely through the Command-Line Interface (CLI). Attendees were guided through kernel interaction, file management, and system setup tasks.

Session 2: Version Control with Git

In the second session, Drona Srivastava introduced Git and GitHub. He demonstrated how to track changes, manage version histories, and restore previous file states. Participants learned to clone repositories, sync changes, and collaborate via branching and merging‚Äîcrucial skills for modern software development.
