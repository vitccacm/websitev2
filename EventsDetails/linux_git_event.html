About the Workshop

This workshop-cum-introduction session was aimed at giving students a strong foundation in Linux and Git & GitHub. Participants explored the fundamentals of Linux, including the Linux kernel, Linux distributions, GNU project, Bash shell, and some basic Linux commands. In the Git & GitHub segment, they learned about version control, Git features, GitHub functionalities, file tracking, collaboration, and how developers can effectively use GitHub for projects.
